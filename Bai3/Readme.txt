## Bài 3 - Nhóm 06
### Đề: Sử dụng bootstrap tạo 1 trang  web  với các yêu cầu sau:
#### Có 4 phần
   - .menu: Là menu chủa các Link bome, about, product, register và form search
   - .slide trinh chiếu (banner)
   - .sản phẩm: chứa các sản phẩm, thông tin 1 sản phẩm: Tên, hình, giá, 2 button: gía và chi tiết
   - .footer
 
#### Yêu cầu giao diện:
   - Thay đổi theo kích thước thiết bị (Responsive)
   - Khi click vào Register, hiện modal chứa form ký thông tin đăng nhập
  tin đăng ký

### Hướng dẫn cài đặt và sử dụng
- Mở file index.html bằng chrome hoặc dùng extension live server của vscode để chạy
- Yêu cầu có kết nối với internet (do dùng cdn)
